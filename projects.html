Dodge

I developed a game using the Unity Game Engine during summer and have since released it on Kongregate as a free to play game. The game is a simple arcade game where the objective is to earn a high score running away from enemies spawning from all over the screen.

Piano Tiles Player

I created a python script that could play the popular game Piano Tiles. This project used the PyAutoGUI to control the mouse and take screenshots and the PIL library to manipulate the screenshots and gather information about the current game state.
